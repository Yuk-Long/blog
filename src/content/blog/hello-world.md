这是我的第一篇博客文章。经过一番折腾，终于成功搭建了一个属于自己的博客。

## 搭建过程

说起来其实并不复杂，主要用到了 **Astro** 这个静态网站框架。它让我可以用 Markdown 写文章，然后自动生成漂亮的网页。

### 用到的技术

- **Astro** — 静态网站框架
- **Markdown** — 编写文章的格式
- **Node.js** — 运行环境

## 代码示例

写一篇技术文章时，代码高亮是必不可少的：

```javascript
// 一段简单的 JavaScript 代码
function sayHello(name) {
  return `你好，${name}！欢迎来到我的博客。`;
}

console.log(sayHello('朋友'));
```

```python
# Python 代码也支持
def fibonacci(n):
    if n <= 1:
        return n
    return fibonacci(n - 1) + fibonacci(n - 2)

print(fibonacci(10))  # 输出 55
```
